# UsedCardList

CarList of used chinese cars

```bash
http://localhost:5173
```

## Вимоги

- [Node.js](https://nodejs.org/) 20.13.1
- [Npm](https://www.npmjs.com/) 10.5.2

# Локальний запуск

Встановлюємо необхідні залежності

```bash
npm install
```

Запускаємо проект локально

```bash
npm run dev
```
